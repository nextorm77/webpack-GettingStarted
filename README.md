# webpack-GettingStarted
## Basic Setup
### node_modules 생성
$ ./npm i
### entry point
$ ./index.html
## Creating a Bundle
### node_modules 생성
$ ./npm i
### 번들 명령어
$ ./npx webpack
### entry point
$ ./dist/index.html
## NPM Scripts
### node_modules 생성
$ ./npm i
### 번들 명령어
$ ./npm run build
### entry point
$ ./dist/index.html
